# 智能投研原型说明

## 原型入口

- `index.html`：B2 Full Pass 高保真 HTML 原型。
- `junior-pass.html`：B2 Junior Pass 初步结构确认稿，仅用于过程留痕。

## 页面清单

| 页面 | 原型位置 | 说明 |
|------|----------|------|
| P01 客户端对话页 | `index.html` 默认客户端模式 | 面向散户 / 投顾的低噪音投研问答 |
| P02 管理端可观测页 | 左下切换到“管理端” | 同步会话并展示右侧 Trace 链路 |
| P03 会话历史视图 | 左侧会话列 | 支持会话切换、新会话临时标题、首条 Query 命名、按列宽省略、拖拽调宽 |
| P04 Trace 详情面板 | 管理端右侧 | 默认摘要，点击内联展开，长内容查看完整 JSON，支持拖拽调宽 |
| P05 本地知识库 / Mock 数据说明页 | 顶部“数据说明” | 展示本地模拟数据与 RAG 状态 |
| P06 系统设置页 | 管理端顶部“系统设置” | 展示模型、Prompt、合规规则配置状态；客户端不展示该入口 |

## 关键组件

- 固定底部输入框：消息区独立滚动，输入框不被历史或消息顶走。
- 自适应消息气泡：短文本短气泡，长文本在最大宽度内换行。
- 会话历史侧边栏：发送首条 Query 前可显示“新对话”，发送后标题按首条 Query 生成，超出列宽自动省略；侧边栏可拖拽调宽。
- 富响应组件：排行表、个股卡、热点摘要、引用来源、风险提示。
- 交互式测算组件：支持买入价、情景价、持仓数量、费率输入，实时计算收益率和盈亏。
- Trace 节点卡：耗时右对齐，点击节点内联展开详情。
- Trace 面板调宽：管理端右侧面板支持拖拽调整宽度。
- 完整 JSON 弹窗：用于查看节点完整结构化数据。

## 交互状态

- 客户端 / 管理端切换。
- 历史会话切换。
- 历史会话搜索：输入关键词后实时过滤会话标题和 Agent 类型，无结果时显示空状态。
- 新建会话：发送前显示“新对话”，发送后标题替换为首条 Query。
- 输入 Query 并生成模拟回答。
- 历史侧边栏拖拽调宽。
- Trace 节点展开 / 收起。
- 查看完整 JSON 弹窗。
- 右侧 Trace 面板拖拽调宽。
- 交互式测算组件实时更新。
- 数据说明 / 系统设置切换；系统设置入口仅在管理端显示。

## Mock 数据结构

原型内置本地模拟数据：

- `conversations`：会话列表、首条 Query、消息、富响应类型、Trace 关联。
- `traces`：节点标题、耗时、摘要、展开详情、完整 JSON。
- 富响应类型：`ranking`、`stock`、`hotspot`、`calculator`、`knowledge`。

生产前端可按 PRD 的数据契约替换为真实接口响应。

## 需要真实 API 的位置

- LLM：通过硅基流动调用 DeepSeek，用于总控 Agent、Sub Agent、回答生成和质检。
- Embedding / Rerank：硅基流动千问，用于本地知识库检索排序。
- LangGraph：后端 Agent 编排，进入后端前必须先补充完整流转图。
- 行情 / 财务 / 公告 / 宏观：MVP 暂不接真实金融 API，后续如接入需补数据源账号、字段和合规说明。

## 必须生产实现的视觉效果

- 输入框固定在对话主区底部。
- 三个滚动容器独立：历史列表、消息区、Trace 面板。
- 会话标题按容器宽度自动省略。
- 历史侧边栏可拖拽调宽，且不影响输入框稳定。
- Trace 节点耗时稳定右对齐。
- Trace 节点默认摘要，点击内联展开，长内容通过 JSON 弹窗查看。
- 右侧 Trace 面板可拖拽调宽，调整时中间对话区和底部输入框保持稳定。
- 富响应组件不能固定死宽高，表格在窄容器内可横向滚动。

## 原型边界

- 本目录 HTML 仅用于设计验证，不作为生产前端源码。
- 生产前端应使用 React + TypeScript + Vite 重建。
- 原型中所有金融数据均为本地模拟数据，不代表真实行情，不构成投资建议。
